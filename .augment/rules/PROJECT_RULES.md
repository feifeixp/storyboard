# AI Director 项目核心规则

**创建时间**: 2024-12-28
**目的**: 避免重复犯错，确保设计质量和通用性

---

## 🎯 核心原则

### 1. 通用工具原则 ⭐⭐⭐

**这是一个通用工具，不是针对特定角色的工具**

✅ **必须支持**：
- 男性、女性、儿童、老年人
- 主角、配角、群众演员
- 各种性格（温柔、强势、聪慧、憨厚等）
- 各种身份（贵族、平民、农民、商人等）
- 各种剧本类型（言情、悬疑、历史、现代等）

❌ **禁止**：
- 局限于"女主角"、"22岁"等特定场景
- 预设角色类型
- 使用特定角色的例子作为标准

---

### 2. 专业判断 vs 硬编码规则 ⭐⭐⭐

**参考分镜生成的成功经验：让LLM像专业人士一样思考**

✅ **正确做法**：
- 教LLM如何做专业判断
- 让LLM像专业造型师/导演一样思考
- 每一步都要求"为什么"
- 通过"自我批判"提升质量

❌ **错误做法**：
- 给LLM一堆硬编码规则
- 规定"必须做什么"或"不能做什么"
- 限制LLM的创造力

**反面案例**：
```
❌ "不要选择碎花裙，因为老气"
❌ "影视剧美学必须白皙细腻"
❌ "禁止使用塑料扣"
❌ "极致美型类必须完美无瑕"
```

**正确案例**：
```
✅ "思考：这个款式是否适合这个角色？为什么？"
✅ "如何在真实和美之间平衡？"
✅ "这个设计是否有魅力？如果不满意，如何改进？"
```

---

### 3. 避免标签化 ⭐⭐⭐

**任何元素都可以用，关键是"如何设计"**

❌ **错误思维**：
```
碎花裙 = 老气
针织开衫 = 老气
塑料扣 = 廉价
```

✅ **正确思维**：
```
碎花裙 + 精致设计 + 好的材质 + 合适的搭配 = 时尚、年轻
针织开衫 + 合适的版型 + 好的颜色 + 精致的细节 = 优雅、有品味
```

**关键**：
- 不限制款式、材质、颜色等元素
- 而是教LLM如何设计得精致、有品质
- 重点是：材质、颜色、版型、细节、搭配

---

### 4. 多样性与独特性 ⭐⭐⭐

**核心理念**：每个角色都应该有独特的视觉特征，但不通过硬编码规则实现

#### 4.1 多样性的矛盾

**矛盾**：
- ✅ 我们需要：角色有独特性，避免千篇一律
- ✅ 我们禁止：硬编码"避免XX元素"
- ❓ 如何解决：在不硬编码的情况下实现多样性？

**解决方案**：
- ✅ 不是限制元素，而是引导思考
- ✅ 不是说"避免XX"，而是问"如何更独特？"
- ✅ 不是禁止重复，而是鼓励突破

#### 4.2 独特性的具体化

**抽象的问法**（效果差）：
```
❌ "这个设计是否独特？"
```

**具体的问法**（效果好）：
```
✅ "这个设计有什么独特的地方？请具体说明至少2个独特特征。"
✅ "如果让观众一眼记住这个角色，你会强调哪个视觉特征？为什么？"
✅ "这个角色与典型的XX角色有什么不同？请具体说明。"
```

**关键**：
- 要求LLM具体说明独特特征
- 要求LLM解释为什么独特
- 要求LLM对比典型设计

#### 4.3 反向思考方法

**错误做法**（硬编码）：
```
❌ "避免深灰色"
❌ "避免针织开衫"
❌ "避免鹅蛋脸"
```

**正确做法**（反向思考）：
```
✅ "除了最常见的选择，是否有更有特色、更能体现角色个性的方案？"
✅ "如果想让这个角色在视觉上更有冲击力，可以在哪些方面突破？"
✅ "作为专业造型师，你会如何避免设计的平庸化？"
```

**关键**：
- 不是禁止某个元素
- 而是鼓励思考更独特的方案
- 让LLM自己判断什么是"平庸"

#### 4.4 专业突破引导

**模拟专业造型师的思考**：
```
✅ "作为专业造型师，你会如何让这个角色在众多角色中脱颖而出？"
✅ "如果这是一部剧的重要角色，如何通过造型让观众过目不忘？"
✅ "从专业角度，这个设计的记忆点在哪里？如果不够突出，如何改进？"
```

**关键**：
- 强调专业判断
- 强调记忆点和辨识度
- 让LLM自己思考如何突破

---

### 5. 历史记录的使用（如果系统提供）⭐⭐

**背景**：
- 用户会连续生成多个角色
- 需要避免角色之间的重复
- 历史记录可以帮助LLM了解"什么是重复"

#### 5.1 历史记录的正确使用

**如果系统提供历史记录**：
```
✅ "最近生成的角色：
   1. 角色A：深灰色针织开衫 + 深棕色裤子
   2. 角色B：深灰蓝色针织开衫 + 牛仔裤
   3. 角色C：深灰色衬衫 + 深蓝色裤子

   思考：如何设计出与以上角色有明显区别的新角色？"
```

**关键**：
- ✅ 提供历史记录作为参考
- ✅ 问"如何有明显区别"
- ❌ 不要说"避免使用深灰色"

#### 5.2 对比思考

**正确的对比引导**：
```
✅ "这个设计与最近生成的角色有什么明显区别？"
✅ "如果观众同时看到这几个角色，能否一眼区分？"
✅ "如何让这个角色更有辨识度？"
```

**错误的对比引导**：
```
❌ "避免使用历史记录中出现的款式"
❌ "不要选择历史记录中的颜色"
```

**关键**：
- 不是禁止重复元素
- 而是引导思考如何区分
- 让LLM自己判断

## 🏗️ 设计方法

### 思维链结构：理解 → 定位 → 设计 → 批判

**每个阶段的核心**：

1. **理解**：深入理解角色、剧本、时代背景
2. **定位**：确定视觉风格、美学方向
3. **设计**：设计具体元素，并解释"为什么"
4. **批判**：自我检查质量，发现问题立即改进

**关键**：
- 每一步都要求"为什么"
- 不是机械执行，而是专业判断
- 通过"自我批判"提升质量

---

### 提问式引导 vs 规定式约束

❌ **规定式约束**（硬编码）：
```
"必须选择真丝、羊绒等高级面料"
"不能选择塑料扣"
"肤色必须白皙细腻"
```

✅ **提问式引导**（专业判断）：
```
"选择什么材质？为什么？（考虑质感、档次、真实性）"
"选择什么颜色？为什么？（考虑角色性格、年龄、时代）"
"这个设计是否精致、有品质？如果不满意，如何改进？"
```

---

## 📊 质量标准

### 通用的评判标准（适用于所有角色）

✅ **好的设计应该**：
1. 有魅力和吸引力（符合影视剧美学）
2. 符合角色定位（基于角色的性格、身份、年龄、故事线）
3. 体现角色性格（通过外貌、服装暗示性格）
4. 精致、有品质（材质、细节、搭配都用心）
5. 真实可信（符合时代背景和角色经历）
6. 有独特性（不千篇一律）

❌ **避免的问题**：
1. 老气、土气、缺乏美感
2. 过度设计、诡异、病态
3. 风格混乱（混用不同风格的特征）
4. 不符合角色定位
5. 缺乏创意、千篇一律

---

### 自我批判的通用问题

✅ **通用问题**（适用于所有角色）：
```
1. 这个设计是否有魅力和吸引力？
2. 是否符合这个角色的定位？（基于角色的具体信息）
3. 是否体现了角色的性格特质？
4. 设计是否精致、有品质？
5. 是否真实可信？
6. 是否有独特性和辨识度？
```

❌ **局限性问题**（只适用于特定角色）：
```
"是否符合女主角的定位？（22岁、有活力、有魅力）" ← 太局限
"是否适合言情剧女主角？" ← 太局限
```

## 📐 人物比例设计原则

### 核心原则

**不要默认所有角色都是8头身！**

人物比例应该根据以下因素综合判断：
1. 年龄（儿童、青少年、成年人、老年人）
2. 美学风格（真实、美化、华丽）
3. 社会阶层（底层、中层、上层）
4. 剧本类型（现实主义、偶像剧、古装剧等）

### 比例分类参考

**按年龄**：
- 儿童（0-12岁）：5-6头身
- 青少年（13-17岁）：6-7头身
- 成年人（18-60岁）：7-8头身（根据其他因素调整）
- 老年人（60+岁）：6.5-7头身

**按美学风格**：
- 真实风格：6.5-7头身（普通人比例）
- 美化风格：7-7.5头身（略微美化）
- 华丽风格：7.5-8头身（理想比例）

**按社会阶层**：
- 底层角色：6.5-7头身（营养一般）
- 中层角色：7-7.5头身（条件较好）
- 上层角色：7.5-8头身（条件优越）

**按剧本类型**：
- 现实主义剧：6.5-7头身
- 偶像剧/言情剧：7.5-8头身
- 古装剧/仙侠剧：7.5-8头身

### 思考问题

在设计人物比例时，应该思考：
- ✅ "这个角色的年龄是多少？应该使用什么比例？"
- ✅ "这个角色的美学风格是什么？（真实、美化、华丽）"
- ✅ "这个角色的社会阶层是什么？经济条件如何？"
- ✅ "这个剧本的类型是什么？需要什么样的视觉呈现？"
- ✅ "综合以上因素，这个角色应该是几头身？"

### 错误做法

❌ **不要**：
- 所有角色都默认8头身
- 不考虑年龄、身份、风格就设定比例
- 底层角色也用8头身（不真实）
- 儿童也用8头身（不符合生理特征）

### 正确做法

✅ **应该**：
- 根据角色特征综合判断比例
- 真实风格的底层角色用6.5-7头身
- 华丽风格的上层角色用7.5-8头身
- 儿童用5-6头身，青少年用6-7头身
- 在输出中说明选择的比例和理由

### 示例

**示例1**：
- 角色：18岁女学生
- 美学风格：美化风格
- 社会阶层：中层
- 剧本类型：校园偶像剧
- **建议比例**：7.5头身
- **理由**：青春偶像剧需要美型呈现，中层家庭营养充足，美化风格适合略高比例

**示例2**：
- 角色：45岁底层工人
- 美学风格：真实风格
- 社会阶层：底层
- 剧本类型：现实主义剧
- **建议比例**：6.5-7头身
- **理由**：现实主义剧强调真实感，底层工人营养一般，真实风格使用普通人比例

**示例3**：
- 角色：8岁儿童
- 美学风格：美化风格
- 社会阶层：中层
- 剧本类型：家庭剧
- **建议比例**：5.5-6头身
- **理由**：儿童的生理特征是头部相对较大，即使是美化风格也不能超过6头身

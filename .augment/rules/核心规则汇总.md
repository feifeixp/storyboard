## 6️⃣ 视频提示词公式

### 核心原则（黄金法则）

1. **使用自然语言描述**：即梦和可灵使用自然语言，不使用权重参数格式
2. **使用摄影术语**：AI视频生成模型训练数据主要是摄影作品，使用摄影术语而非电影分镜术语
3. **语言简洁**：句子结构简单，避免复杂从句
4. **动态适度**：优先简单动作（走路、挥手、微笑），避免大幅度动作（跑酷、打斗）
5. **风格一致**：提示词中必须强调风格保持
6. **时长匹配**：画面内容必须在指定时长内可完成

---

### 提示词技巧（进阶要点）

1. **动作具象化**：不说"愤怒"，说"猛地握紧拳头，指节泛白"
2. **光影具象化**：不说"光线变化"，说"顶光逐渐增强，阴影向下延伸"
3. **动态量化**：不说"快速移动"，说"0.5秒内冲向镜头"
4. **风格强化**：每个提示词都加风格标签（如"赛博朋克风格"）
5. **运镜明确**：必须指定运镜方式（推拉摇移跟升降）
6. **时长控制**：明确指定视频时长（如"5秒视频"）

---

### I2V（图生视频）公式

```
风格保持 + 情绪调节 + 主体动作（具象化+量化） + 运镜方式 + 光影变化（具象化） + 时长
```

**示例**：
```
水墨风格保持不变，仕女缓慢抬起右手，轻执茶杯，
镜头从人物侧面缓慢推进至手部特写，
背景纱帘随微风轻微飘动，
暖光从窗棂渗入，在人物衣袂上形成渐变光影
```

---

### 首尾帧生视频（Keyframe）公式

```
从首帧到尾帧，[过渡方式] + [核心元素变化细节] + [速度节奏] + [光影过渡] + [风格保持] + [时长]
```

**示例**：
```
从首帧到尾帧，人物身体匀速覆盖机械装甲，
装甲先从肩部生成再向手臂躯干腿部延伸，
金属纹理逐步清晰，眼睛变为蓝色发光体，
保持整体姿势不变，光影从自然光转为冷色调机械光，
全程无风格断层，8秒
```

### 能力边界

**✅ AI擅长的动作**：
- 简单肢体动作（挥手、转身、点头、走路）
- 表情变化（微笑→严肃、平静→愤怒）
- 物体移动（飘动、旋转、坠落）
- 光影变化（明暗、颜色、强度）
- 简单运镜（推拉摇移）
- 原有框架内扩展（在原图构图内添加动态）
- 5-10秒时长（最佳时长范围）

**❌ AI难以处理的动作**：
- 复杂肢体动作（打斗、舞蹈、翻滚、跑酷）
- 多人互动（拥抱、握手、对打）
- 精细手部动作（弹琴、写字、打字）
- 口型同步（说话、唱歌）
- 物理规律复杂的动作（液体流动、布料飘动、球类弹跳）
- 突破原图空间框架（如原图室内，难以扩展为室外全景）
- 超过10秒时长（易出现节奏混乱）

---

### 负面提示词（Negative Prompt）

**必须避免的描述**：
- ❌ 模糊、失焦、低质量
- ❌ 变形、扭曲、比例失调
- ❌ 多余的肢体（多手、多脚）
- ❌ 文字、水印、logo
- ❌ 画面抖动、闪烁
- ❌ 风格偏移、风格断层（视频特有）
- ❌ 动作卡顿、不连贯（视频特有）
- ❌ 背景额外元素新增（视频特有）
- ❌ 构图框架突变（视频特有）

**标准负面提示词模板**：
```
blurry, out of focus, low quality, distorted, deformed,
extra limbs, extra fingers, watermark, text, logo,
shaky, flickering, artifacts,
style shift, style inconsistency, choppy motion, jerky movement,
background elements appearing, composition frame change
```

### 透视缩短（foreshortening）规则

**核心概念**：
- 躺倒的人体接近**正方形**，而非长方形
- 大脑会"脑补"成更长的形态，需要强调身体角度符合透视规律

**提示词关键词**：
- 中文：透视缩短，身体压缩成接近正方形，肩膀和膝盖呈自然透视角度
- 英文：foreshortened body, body compressed into square shape, shoulders and knees at natural perspective angles

## 8️⃣ 分镜连续性三原则

### 1. 空间连续性（180度轴线规则）

**核心原则**：镜头始终在"两主要元素连线同侧"

当场景中存在两个核心叙事元素（如"主角A"与"目标B"、"角色与关键道具"）时，需想象二者间存在一条隐形"连线"。所有连续镜头的摄影机，必须固定在这条连线的**同一侧**拍摄，不可突然越至另一侧。

**示例**：
- 若第一镜从"主角（左）- 反派（右）"连线的左侧拍摄，后续镜头需始终保持在左侧
- 若突然越至右侧，会导致观众对角色位置、动线产生认知混乱
- 就像观看足球比赛时镜头突然反切，分不清攻防方向

### 2. 视觉中心连续性

**核心原则**：上一镜视觉焦点，下一镜优先承接

镜头切换时，需确保上一镜的"视觉中心"（如角色的眼睛、关键道具、动作终点），在下一镜中以"承接式构图"出现——或直接作为新镜头的视觉焦点，或通过"引导线"（如角色视线、道具延伸方向）自然过渡。

**示例**：
- 上一镜特写"主角看向右侧的手"
- 下一镜需优先展示"右侧手的动作"或"手所指向的目标"
- 避免视觉焦点跳脱，让观众视线"有迹可循"

### 3. 动作与视线连续性

**核心原则**：动作衔接顺承，视线方向统一

角色的"物理动作"与"视线方向"需保持逻辑顺承：

**动作衔接**：
- 上一镜"抬手"，下一镜需延续为"握拳"或"伸手取物"
- 不可突然跳转为"垂手"

**视线衔接**：
- 上一镜角色"看向画右"，下一镜需展示"画右的目标"（如反派、道具）
- 而非突然切至"画左的无关元素"

这种顺承并非机械重复，而是通过"动作拐点""视线引导"强化剧情逻辑——即使镜头景别、角度变化，核心动作与视线的"因果关系"需清晰可见。

**详细说明**：请查阅 [分镜设计连续性三原则](分镜设计连续性三原则.txt)

## 🔟 POV镜头使用指南

### 定义

**POV镜头**（Point of View Shot）= **主观视角** = 从角色眼睛看出去的第一人称视角

### 标准格式

```
"主观视角，[目标物体/角色的状态和位置]"
```

### 正确示例

| 场景 | 提示词 |
|------|--------|
| 发现敌人 | "主观视角，敌人正面站立在3米外，表情凶狠，手持武器" |
| 追逐场景 | "主观视角，人物背对镜头向前奔跑，距离5米，披风飘动" |
| 探索空间 | "主观视角，走廊向前延伸20米，尽头是发光的蓝色门，两侧墙壁布满电路纹路" |
| 发现线索 | "主观视角，神秘符号刻在墙上，发出微弱红光，距离1米" |

### 关键要点

1. **必须使用"主观视角"开头** - 这是AI能理解的标准术语
2. **禁止提及"眼睛"** - 会被误解为画眼睛特写
3. **描述目标物体，而非视角起点** - 重点是"看到什么"，不是"从哪里看"
4. **包含距离信息** - 帮助AI理解空间关系（如"3米外"、"5米"）

### 使用频率

- **建议占比**: ≤5%（30个镜头最多1-2个）
- **原因**: POV镜头代入感强，但过多会导致视角单调
- **搭配建议**: 与3/4正面、正侧面等常规角度交替使用

### 适用场景

| 场景类型 | 情绪效果 |
|---------|---------|
| 发现关键线索 | 强代入感、揭秘感 |
| 惊恐/震撼时刻 | 强烈恐惧、压迫感 |
| 追逐中的威胁视角 | 紧张、追逐感 |
| 探索未知空间 | 好奇、探索感 |

### 禁用词列表

以下词汇会导致画面出现眼睛特写：
- "从眼睛看出去"
- "眼睛位置"
- "从角色眼睛"
- "从[角色名]眼睛"（如"从晋安眼睛"、"从林溪眼睛"等）
- "角色视线看出去"

**替换建议**: 全部改为"主观视角"

**详细说明**：请查阅 [POV镜头使用指南](POV镜头使用指南.md)

## 1️⃣1️⃣ 短剧漫剧剧本核心要求

### 节奏要求

| 时间节点 | 要求 |
|---------|------|
| 黄金10秒 | 建立情境 + 抛出钩子（直接切入冲突场景） |
| 30秒 | 爆发核心冲突（明确主角目标） |
| 1分钟 | 一个小转折（插入小高潮或反转） |
| 场景时长 | 10-20秒短场景，快速切换 |
| 结尾 | 卡在冲突最高点（强制吸引追更） |

### 爽点设计

- **15秒一个小爽点**：每集至少1次"危机-解决"闭环（如"被羞辱→激活金手指→秒反杀"）
- **反差感强化**：身份反差（"扫地小厮=仙界帝尊"）、实力反差（"废柴秒变王者"）
- **情绪曲线起伏**：平静→冲突陡升→小挫折→情绪顶峰

### 内容与表达

- **角色标签鲜明**：3秒让观众记住（外貌标志+标志性动作+专属台词）
- **视觉化转译**：把内心戏转化为"动作+对话"（不说"她很生气"，而是"她猛地拍桌，指节泛白：'你敢骗我？'"）
- **台词精炼有力**：多用短句、感叹句，避免冗长叙述
- **支线彻底精简**：聚焦核心主线，砍掉无关次要情节

### 结构与适配

- **单集独立闭环**：1-3分钟（300-800字），做到"3秒吸睛+冲突升级+卡点悬念"
- **三段式结构**：开端抛悬念→过程加深冲突→结尾反转/留钩
- **分镜思维预埋**：明确标注景别（特写、中景）、动作细节、特效需求（如"玉佩发光特效"）

**详细说明**：请查阅 [短剧漫剧剧本的核心要求](短剧漫剧剧本的核心要求.txt)

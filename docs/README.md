## 🎯 功能概述

智能补充功能可以自动从剧本中提取角色信息，补充角色档案中缺失的字段，包括：

- ✅ **外观描述** - 详细的三段式外观描述
- ✅ **身份演变** - 角色身份的发展轨迹
- ✅ **经典台词** - 最能代表角色的台词
- ✅ **能力进化** - 角色能力的成长过程
- ✅ **多形态图鉴** - 不同造型的详细记录

## 📖 使用场景

### 场景1：新角色快速建档
1. 创建角色，只填写姓名和性别
2. 上传剧本
3. 点击智能补充
4. 系统自动填充所有信息

### 场景2：补充多形态信息
1. 角色已有基本信息
2. 剧本中有多个造型描述
3. 点击智能补充
4. 系统自动识别并创建多个形态

### 场景3：优化现有角色
1. 角色信息不够详细
2. 点击智能补充
3. 系统补充缺失字段
4. 手动优化细节

## ⚠️ 注意事项

1. **API Key 配置**
   - 项目已配置好 `VITE_OPENROUTER_API_KEY`（在 `.env.local` 文件中）
   - 如需更换 API Key，编辑 `.env.local` 文件并重启服务器

2. **剧本质量**
   - 剧本内容越详细，补充效果越好
   - 建议包含对话和动作描述
   - 按集数组织内容

3. **手动优化**
   - 补充后建议检查结果
   - 可以手动调整不准确的部分
   - 可以添加个性化细节

## 🐛 常见问题

### API Key 无效
- 检查 `.env.local` 文件是否存在
- 确认 `VITE_OPENROUTER_API_KEY` 配置正确
- 重启开发服务器

### 补充失败
- 检查 API 余额
- 确认网络连接
- 查看剧本内容

### 结果不准确
- 优化剧本描述
- 手动编辑结果
- 重新触发补充

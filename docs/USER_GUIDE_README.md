## 🛠️ 提示词优化助手使用指南

### 功能介绍

提示词优化助手是集成在HTML手册底部的智能工具，可以帮助你快速生成符合规范的提示词。

### 使用步骤

1. **打开工具**：
   - 点击页面底部的"🛠️ 提示词优化助手"标题栏
   - 工具会展开，显示完整界面

2. **输入描述**：
   - 在左侧文本框中输入你的画面描述
   - 例如："一个女孩站在教室窗边，阳光从左侧照入，温暖的午后光线"

3. **优化提示词**：
   - 点击"✨ 优化提示词"按钮
   - 工具会自动分析你的输入

4. **查看结果**：
   - **中文提示词**：符合即梦/可灵规范的中文提示词
   - **英文提示词**：符合Banana Pro规范的英文提示词
   - **技术建议**：景别、角度、字数等建议
   - **违规检测**：检测是否包含禁用术语

5. **复制使用**：
   - 点击"复制"按钮，一键复制提示词
   - 粘贴到AI图像生成工具中使用

### 功能特性

**智能分析**：
- ✅ 自动检测景别（特远景、远景、全景、中景、近景、特写等）
- ✅ 自动检测视角高度（鸟瞰、俯拍、平视、仰拍、虫视等）
- ✅ 自动检测人物朝向（正面、3/4正面、侧面、背面等）

**违规检测**：
- ✅ 检测权重参数格式（如 `(extreme long shot:1.3)`）
- ✅ 检测电影分镜术语（如 `3/4 front view`）
- ✅ 检测POV镜头禁用词（如 `从眼睛看出去`）

**技术建议**：
- ✅ 景别选择建议
- ✅ 角度组合建议
- ✅ 字数统计和优化建议
- ✅ 常见错误提醒

## 🖨️ 打印和导出

手册支持打印和导出为PDF：

1. **打印**：
   - 在浏览器中按 `Ctrl+P`（Windows）或 `Cmd+P`（Mac）
   - 选择打印机或"另存为PDF"
   - 打印样式已优化，会自动隐藏侧边栏和工具栏

2. **导出PDF**：
   - 使用浏览器的"打印"功能
   - 选择"另存为PDF"
   - 保存到本地

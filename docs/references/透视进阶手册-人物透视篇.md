## 第一章：物体阴影的基础原理

### 1.1 阴影三要素

绘制阴影需要三个基本要素：

| 要素 | 说明 |
|------|------|
| **光源** | 光线的来源（太阳、灯泡、蜡烛等） |
| **受光物体** | 被光线照射的物体 |
| **投影面** | 阴影投射到的表面（地面、墙壁等） |

### 1.2 光源类型与阴影特性

**局部光源（Local Light）**：
- 例：灯泡、蜡烛、路灯
- 距离物体较近
- 阴影从光源基点向外辐射

**远距离光源（Sun）**：
- 距离约1.5亿公里
- 光线近乎平行
- 所有阴影方向相同

### 1.3 阴影投射的基本方法

**书中经典案例（图1.1-1.2）**：

```
步骤1：确定光源位置和光源在地面的投影点A
步骤2：从A点画一条线穿过物体底部B点，向远处延伸
步骤3：从光源画一条线穿过物体顶部，直到与AB延长线相交
步骤4：交点就是阴影的终点
```

**关键规则**：
```
✅ 物体顶边与其投影阴影始终平行
✅ 多个物体的阴影从光源投影点向外辐射
✅ 光源越远，阴影边缘越模糊（图1.9）
```

### 1.4 复杂形状的阴影

**开口物体的阴影（图1.8）**：
```
1. 将开口的垂直边延伸到地面（创建点1B和2B）
2. 从光源基点穿过1B和2B画线
3. 从光源穿过开口顶部的点，找到交点
4. 连接形成阴影轮廓
```

**字母E的阴影（图1.10）**：
```
关键：使用一条从上到下穿过字母的垂直参考线
找到所有水平线与垂直线的交点
从光源穿过这些交点投射到地面
```

## 第二章：人物透视基础

### 2.1 人体比例

**黄金比例（图2.4-2.5）**：
```
标准英雄/运动体型 = 8个头高
```

**比例分配**：
| 部位 | 头数 |
|------|------|
| 头部 | 1 |
| 下巴到胸部 | 1 |
| 胸部到腰部 | 1 |
| 腰部到胯部 | 1 |
| 胯部到膝盖 | 2 |
| 膝盖到脚底 | 2 |

### 2.2 透视中的人体

**关键原则**：
```
✅ 人体可以简化为几何体（圆柱、方块）
✅ 关节处的旋转创造动态姿态
✅ 重心线决定平衡感
```

**透视缩短效果**：
- 朝向观众的手臂/腿会显得更短
- 远离观众的部分会显得更长
- 使用椭圆表示透视中的关节

### 2.3 不同角度的人体

**仰视（Low Angle）**：
```
- 下巴、胸部底面可见
- 肩膀看起来更宽
- 人物显得高大、有威压感
```

**俯视（High Angle）**：
```
- 头顶、肩膀顶面可见
- 腿部缩短
- 人物显得渺小、脆弱
```

**极端角度（Extreme Angles）**：
```
- 三点透视应用于人体
- 身体各部分向消失点汇聚
- 创造强烈的视觉冲击
```

## 第三章：人物阴影

### 3.1 头部光影

**光从正上方（图3.1-3.11）**：
```
效果：非常戏剧性
- 眼窝深陷阴影
- 鼻子下方形成三角阴影
- 下巴下方阴影加重
应用：悬疑、恐怖、审判场景
```

**头部简化为几何体（图3.12-3.14）**：
```
技巧：
1. 将头部简化为平面几何体
2. 确定每个平面与光源的关系
3. 根据平面朝向确定明暗

"绘画实际上是一种翻译练习。画出的东西永远不是
'现实本身'，而是对观察到的、被认为关键的内容
的诠释。"
```

### 3.2 面部光影的处理技巧

**顶光处理（图3.36-3.37）**：
```
问题：极端顶光会让面部特征消失
解决方案：
1. 在眼睛区域保留一点光线，让表情可见
2. 使用反射光照亮下巴区域
3. 或者大胆使用纯黑，形成头骨般的效果

注意：女性面部阴影不要像胡子，
可在下巴区域添加反射光
```

**背光/轮廓光（图3.67-3.69）**：
```
效果：
- 脸部两侧有光边
- 面部中心在阴影中
- 强调轮廓而非表情

三种处理方式：
1. 面部用灰色 → 表情可读
2. 面部纯黑 + 过曝背景 → 剪影效果
3. 纯黑背景 → 突出边缘光
```

**底光（图3.70-3.71）**：
```
效果：诡异、不安
原因：光源通常在上方，底光是"反常"的
应用：恐怖场景、超自然元素
技巧：可以用硬边处理，创造雕刻感
```

### 3.3 头部转动与一致性

**书中案例（图3.43-3.66）**：
```
关键：光源不变时，不同角度的头部阴影必须一致

步骤：
1. 从真实参考（照片/模特）开始
2. 简化形状，创造更直接的视觉效果
3. 当大部分面部在阴影中时，选择：
   - 使用灰色让表情可见
   - 或大胆使用黑白对比
```

## 第四章：服装褶皱

### 4.1 褶皱的基本原理

**褶皱产生的原因**：
```
1. 重力：垂直向下的拉力
2. 张力：拉伸的力
3. 压缩：挤压的力
4. 扭转：旋转的力
```

**褶皱的方向规则（图4.26-4.30）**：
```
✅ 褶皱从受力点向外辐射
✅ 褶皱指向身体平面变化的位置
✅ 褶皱帮助定义身体的结构和体积
```

### 4.2 褶皱绘制技巧

**选择性描绘（图4.31-4.33）**：
```
❌ 不要画出每一条可见的褶皱
✅ 选择能定义形状、体积和动态的褶皱
✅ 保持画面的图形化，便于"快速阅读"
```

**阴影处理（图4.34-4.39）**：
```
方法1：纯色块阴影
方法2：定义核心阴影 + 反射光区域用平行线
笔触方向：跟随褶皱方向，帮助表现体积
```

### 4.3 布料与动态

**运动中的布料（图4.26-4.29）**：
```
- 快速运动：褶皱向运动相反方向飘动
- 武术动作：褶皱帮助表现动态张力
- 褶皱发射点（灰色标注）是关键
```

## 第五章：渲染思路与执行

### 5.1 从照片到绘画的翻译

**书中核心理念**：
```
"艺术作品中的内容是一种幻想。为了创造有效的幻想，
重要的是综合所观察到的内容，'消化'从观察过程中
提取的信息。"
```

**简化与组织**：
```
步骤1：观察真实场景/照片
步骤2：确定视平线和消失点
步骤3：简化形状，去除干扰元素
步骤4：强化黑白对比，创造图形化效果
```

### 5.2 实战技巧

**树木与建筑（图5.3-5.5）**：
```
- 在背景树木中使用纯黑，让前景建筑"跳出"
- 简化窗户细节，利用明暗对比
- 寻找大胆使用墨水的机会
```

**纹理线条（图5.3）**：
```
在墙壁上添加纹理线条：
- 帮助建立透视感
- 增强结构感和深度
- 必须基于正确的透视网格
```

**轮胎与体积（图5.8）**：
```
在轮胎顶部保留一点光：
- 建立体积感
- 纯黑轮胎无法表达圆形
```

## 第八章：人物透视变形详解

### 8.1 不同角度下的人物变形

**极端仰拍（Worm's Eye View）**：
```
可见部分：
- 下巴轮廓锋利突出
- 鼻孔可能隐约可见
- 肩膀呈宽大倒三角
- 胸部底面可见
- 腿部透视缩短

不可见部分：
- 头顶
- 肩膀顶面
```

**极端俯拍/鸟瞰（Bird's Eye View）**：
```
可见部分：
- 头顶和发型
- 肩膀顶面
- 背部轮廓
- 脚部和地面

不可见部分：
- 下巴
- 胸部正面
- 面部表情（除非低头）
```

### 8.2 透视缩短效果

**朝向镜头的肢体**：
```
- 手臂伸向镜头 → 手部放大，手臂缩短
- 拳头冲向镜头 → 拳头巨大，手臂几乎不可见
- 脚踢向镜头 → 脚底放大，腿部缩短
```

**远离镜头的肢体**：
```
- 手臂远离镜头 → 手部缩小，手臂看起来更长
- 人物背向跑去 → 背部放大，腿部缩小
```

## 第九章：阴影投射进阶

### 9.1 阴影方向与光源角度

| 光源位置 | 本影位置 | 投影方向 | 情绪效果 |
|----------|----------|----------|----------|
| 正上方 | 眼窝、鼻下、下巴下 | 正下方 | 阴森、审判 |
| 左上方 | 右脸、右侧身体 | 右下方 | 戏剧性 |
| 右上方 | 左脸、左侧身体 | 左下方 | 戏剧性 |
| 正下方 | 额头、鼻上、脸颊上 | 正上方 | 诡异、恐怖 |
| 正后方 | 整个正面 | 朝向镜头 | 剪影、神秘 |

### 9.2 反射光的应用

**反射光**（Reflected Light）是从其他表面反弹回来的光：
```
应用场景：
- 顶光时，地面反射光照亮下巴下方
- 侧光时，对面墙壁反射光照亮阴影面
- 雪地/水面场景中反射光特别强

效果：
- 软化阴影边界
- 增加体积感
- 让黑暗区域不至于完全死黑
```

### 9.3 边缘光/轮廓光

**边缘光**（Rim Light）从背后照亮物体边缘：
```
效果：
- 将主体与背景分离
- 创造发光的轮廓线
- 增加三维立体感
- 可用于正面全黑的剪影效果

提示词：
中文：背光轮廓光勾勒边缘，头发和肩膀发出金色光边
英文：(rim light:1.3), backlight outlining edges, golden glow on hair and shoulders
```

## 第十章：布料物理学

### 10.1 褶皱产生的四种力

| 力的类型 | 产生位置 | 褶皱特征 | 示例 |
|----------|----------|----------|------|
| **重力** | 悬挂点 | 垂直向下的褶皱 | 披风、裙摆 |
| **张力** | 拉伸点 | 从拉伸点向外辐射 | 弯曲的肘部、膝盖 |
| **压缩** | 挤压点 | 密集的褶皱堆积 | 弯腰时的腰部 |
| **扭转** | 旋转点 | 螺旋状褶皱 | 扭转的手臂、身体 |

### 10.2 关节处的褶皱规律

```
规律：
- 褶皱从关节内侧向外辐射
- 关节弯曲角度越大，褶皱越多
- 关节外侧的布料被拉紧，褶皱少
- 关节内侧的布料被压缩，褶皱密集

示例（弯曲的肘部）：
- 肘内侧：密集的压缩褶皱
- 肘外侧：少量拉伸纹理
```

### 10.3 运动中的布料

```
静止状态：
- 布料垂直向下
- 褶皱由重力决定

运动状态：
- 布料向运动相反方向飘动
- 运动越快，飘动越剧烈
- 停止瞬间，布料会有"滞后"飘动

提示词：
中文：披风向后上方飘动，褶皱从肩膀辐射，形成流动的S形曲线
英文：cape billowing backward and upward, folds radiating from shoulders, forming flowing S-curve
```

## 第十一章：渲染风格与情绪

### 11.1 线条风格

| 风格 | 特点 | 情绪效果 | 应用 |
|------|------|----------|------|
| **硬边线条** | 锐利、清晰 | 紧张、冲突 | 动作场景 |
| **软边线条** | 模糊、渐变 | 柔和、梦幻 | 回忆、浪漫 |
| **粗线条** | 厚重、力量 | 强烈、戏剧 | 关键时刻 |
| **细线条** | 精致、轻盈 | 细腻、优雅 | 静态、对话 |

### 11.2 黑白对比度

| 对比度 | 效果 | 应用场景 |
|--------|------|----------|
| **高对比** | 戏剧性、紧张 | 冲突、高潮、恐怖 |
| **低对比** | 平和、日常 | 对话、铺垫 |
| **纯黑白** | 极端、图形化 | 剪影、标志性时刻 |
| **灰度丰富** | 细腻、真实 | 情感细腻场景 |

## 📚 前言

本手册是《分镜脚本设计新手手册》的进阶补充，专注于**场景透视**的深度理论与实战应用。透视不仅是技术工具，更是**叙事语言**——正确的透视选择能够强化情绪、引导视线、创造戏剧张力。

## 第一章：透视的基础原理

### 1.1 视觉的本质

> "透视是在二维平面上表示三维空间的方法。" —— Marcos Mateu-Mestre

**核心概念**：

| 术语 | 定义 | 作用 |
|------|------|------|
| **视线高度** | 观察者眼睛的高度 | 决定地平线位置 |
| **地平线/视平线** | 与观察者视线等高的水平线 | 所有水平消失点都在此线上 |
| **消失点(VP)** | 平行线汇聚的点 | 创造纵深感 |
| **视觉中心** | 观察者直视的点 | 画面的焦点区域 |

**关键规则**：
```
✅ 视线以下的物体 → 从上往下看（看到顶面）
✅ 视线以上的物体 → 从下往上看（看到底面）
✅ 物体越远 → 看起来越小（透视缩小）
✅ 平行线 → 汇聚到同一消失点
```

---

### 1.2 观察者与物体的相对位置

**案例：立方体的视觉变化**

书中阿丽亚娜(Arianna)观察立方体的经典案例：

```
情况1：立方体在视线高度
→ 只能看到正面，看不到顶面或底面

情况2：立方体在视线以下（放在地面）
→ 从上往下看，能看到顶面

情况3：立方体在视线以上（悬挂在天花板）
→ 从下往上看，能看到底面
```

**分镜应用**：
- 低机位（仰拍）→ 角色显得高大、有威胁感
- 高机位（俯拍）→ 角色显得渺小、脆弱
- 平视 → 客观、中立

## 第二章：一点透视（单点透视）

### 2.1 基本原理

**特点**：
- 只有**一个消失点**
- 物体正面朝向观察者
- 所有后退的平行线汇聚到同一点

**适用场景**：
- 走廊、隧道、街道
- 铁轨、公路
- 室内正面视角

### 2.2 一点透视的情绪表达

| 消失点位置 | 情绪效果 | 应用案例 |
|------------|----------|----------|
| **正中央** | 稳定、庄严、对称、神圣 | 教堂、宫殿、仪式场景 |
| **偏左/偏右** | 不安、动态、暗示方向 | 追逐、逃跑 |
| **偏上** | 压迫、下沉感 | 地牢、监狱 |
| **偏下** | 开阔、上升感 | 天空、希望 |

### 2.3 书中经典案例

**案例1：鬼屋走廊（图3-28）**
```
技巧：
- 单点透视创造深度感
- 墙壁和门的连续层叠增强纵深
- 硬木地板的横梁隐藏在阴影中
- 效果：神秘感、迷宫般的层次
```

**案例2：战士剪影（图3-30 vs 图3-31）**
```
图3-30：
- 所有线条指向人物
- 头部背景更亮
- 居中位置
- 效果：超凡脱俗、神圣感

图3-31：
- 平直墙壁作为背景
- 消除纵深感
- 效果：坚实、不可逾越
```

**案例3：公寓楼（图3-32 vs 图3-33）**
```
低视角（图3-32）：
- 对称构图
- 只能看到公寓和前院
- 效果：坚实、封闭

高视角（图3-33）：
- 能看到远处城市
- 地理背景更丰富
- 效果：开阔、叙事性强
```

## 第三章：两点透视

### 3.1 基本原理

**特点**：
- **两个消失点**在地平线两端
- 物体呈角度面向观察者
- 能同时看到物体的两个侧面

**何时使用**：
- 当观察者相对于物体移动或旋转时
- 需要展示建筑的立体感
- 创造更动态的画面

### 3.2 两点透视的视觉效果

```
完美对称的两点透视：
- 两个消失点与中心等距
- 物体两个面大小相等
- 效果：平衡、稳定

不对称的两点透视：
- 一个消失点更近
- 一个面更大、更正面
- 效果：动态、有方向性
```

### 3.3 消失点距离与画面效果

| 消失点距离 | 效果 | 应用 |
|------------|------|------|
| **很远（画面外）** | 自然、舒适 | 日常场景 |
| **较近** | 夸张、戏剧性 | 动作场景 |
| **极近** | 扭曲、不安 | 恐怖、超现实 |

## 第四章：三点透视

### 4.1 基本原理

**特点**：
- 第三个消失点在**垂直方向**
- 向上（天顶）→ 仰视效果
- 向下（最低点）→ 俯视效果

**何时使用**：
- 极端仰拍（摩天大楼、巨人）
- 极端俯拍（悬崖、高空）
- 需要强烈视觉冲击时

### 4.2 向上三点透视（仰视）

```
第三消失点在天顶：
- 垂直线向上汇聚
- 建筑物向上收窄
- 效果：崇高、压迫、渺小感
```

**应用场景**：
- 超级英雄俯视镜头
- 摩天大楼仰拍
- 反派的威压感

### 4.3 向下三点透视（俯视）

```
第三消失点在最低点：
- 垂直线向下汇聚
- 建筑物向下收窄
- 效果：眩晕、危险、上帝视角
```

**应用场景**：
- 悬崖边缘俯瞰
- 高楼坠落
- 鸟瞰城市

### 4.4 书中经典案例

**案例：三点透视建筑（图6-15 vs 图6-16）**
```
图6-15（仰视）：
- 从下面看建筑
- 天顶消失点
- 效果：建筑高耸入云

图6-16（俯视）：
- 从上面看建筑
- 最低点消失点
- 效果：俯瞰全景
```

## 第五章：多重消失点

### 5.1 何时需要多个消失点

**情况1：不同方向的物体**
- 每组平行线有自己的消失点
- 旋转的物体有不同的消失点

**情况2：复杂建筑群**
- 不规则布局的塔楼
- 不同角度的墙壁

### 5.2 书中经典案例

**案例：中世纪城堡（图5-10）**
```
技巧：
- 多个消失点表示不同方向的塔楼
- 保持比例一致性
- 使用参考线A-B确定高度
```

**案例：模块化建筑（图5-21）**
```
技巧：
- 不同方向的模块创造张力
- 光线集中在特定区域
- 效果：凸显居住者的个性
```

## 第六章：倾斜面与楼梯

### 6.1 倾斜面的消失点

**关键规则**：
```
倾斜面的消失点不在地平线上！
- 向上倾斜 → 消失点在地平线上方
- 向下倾斜 → 消失点在地平线下方
- 倾斜角度相同 → 消失点与地平线距离相同
```

### 6.2 楼梯的绘制

**书中方法（图7-14~7-15）**：
```
步骤1：确定楼梯的起点和终点
步骤2：将梯段分成相等的部分（踏板数量）
步骤3：将高度分成相等的部分（立板数量）
步骤4：连接对应的点形成台阶
```

**四面楼梯（图7-19~7-23）**：
```
- 四个不同方向的斜面
- 四个不同的消失点
- 中央消失点与各踏步消失点距离相等
```

## 第七章：构图与透视的结合

### 7.1 构图模式总结

书中第十二章"构成笔记"总结了多种构图模式：

| 构图类型 | 特点 | 情绪效果 |
|----------|------|----------|
| **V形** | 线条向中心汇聚 | 聚焦、引导 |
| **H形** | 水平连接两个主体 | 动态、对抗 |
| **Z形** | 对角线下降 | 流动、叙事 |
| **U形** | 包围主体 | 保护、框架 |
| **螺旋形** | 旋转引导视线 | 动感、深度 |
| **对角线** | 斜向分割画面 | 张力、不稳定 |

### 7.2 明暗与透视的配合

**书中案例分析**：

```

- 透视：单点透视，线条指向人物

- 效果：双重聚焦

图5-22：
- 透视：多重消失点
- 明暗：光线集中在特定区域
- 效果：层次分明
```

## 第八章：徒手绘制技巧

### 8.1 基本练习

**书中推荐练习（第十一章）**：

```
练习1：同心圆
- 从大到小画同心圆
- 放松手腕，不要紧张
- 重复1000遍比100遍好

练习2：矩形内椭圆
- 在矩形内徒手画椭圆
- 练习手眼协调

练习3：汇聚平行线
- 画向同一消失点汇聚的线
- 保持间距均匀
```

### 8.2 测量与比例

**书中方法**：
```
1. 确定第一个参考点（如头部大小）
2. 用这个参考测量其他部分
3. 利用负空间确认比例
4. 画出连接关键点的辅助线
```

## 第九章：实战应用清单

### 9.1 分镜透视选择指南

| 场景类型 | 推荐透视 | 原因 |
|----------|----------|------|
| 走廊/隧道 | 一点透视 | 强调纵深 |
| 建筑外观 | 两点透视 | 展示立体感 |
| 摩天大楼 | 三点透视（仰） | 强调高度 |
| 悬崖俯瞰 | 三点透视（俯） | 创造眩晕感 |
| 复杂城市 | 多重消失点 | 表现复杂性 |
| 楼梯场景 | 倾斜面透视 | 正确表现斜面 |

### 9.2 常见错误检查表

```
❌ 平行线不汇聚到同一消失点
❌ 地平线位置不一致
❌ 物体比例随距离变化不正确
❌ 仰拍时看到了物体顶面
❌ 俯拍时看到了物体底面
❌ 倾斜面消失点放在地平线上
```

## 第十一章：透视与镜头情绪对照表

### 11.1 相机角度 × 透视类型 × 情绪效果

| 相机角度 | 透视类型 | 情绪效果 | 适用场景 |
|----------|----------|----------|----------|
| **极端仰拍** | 三点透视向上 | 崇高、压迫、威胁 | 反派登场、巨物、摩天大楼 |
| **仰拍** | 两点透视 | 英雄感、力量 | 主角决心、行动时刻 |
| **平视** | 一/两点透视 | 客观、中立 | 对话、日常 |
| **俯拍** | 两点透视 | 弱小、被监视 | 角色失败、孤独 |
| **鸟瞰** | 三点透视向下 | 渺小、全知 | 展示全局、角色孤立 |
| **虫视** | 三点透视向上 | 极端压迫、恐惧 | 巨物、危机、怪物 |

### 11.2 消失点位置 × 画面情绪

**一点透视消失点位置**：
| 消失点位置 | 情绪效果 | 应用 |
|------------|----------|------|
| 画面中央 | 稳定、庄严、宿命感 | 教堂、仪式、最终对决 |
| 偏向左侧 | 向左引导、离开感 | 角色离开、追忆 |
| 偏向右侧 | 向右引导、前进感 | 角色前进、探索 |
| 偏向上方 | 压迫、下沉感 | 地牢、监狱、绝望 |
| 偏向下方 | 开阔、上升感 | 天空、希望、解脱 |

### 11.3 地平线位置 × 画面氛围

| 地平线位置 | 视觉效果 | 情绪 | 应用场景 |
|------------|----------|------|----------|
| 上1/3处 | 地面占主导 | 接地、现实、压迫 | 战场、废墟、追逐 |
| 中央 | 平衡 | 中性、平淡 | 避免使用，缺乏张力 |
| 下1/3处 | 天空占主导 | 开阔、自由、希望 | 英雄抬头、觉醒 |
| 极低 | 几乎全是天空 | 渺小、宏大 | 仰望建筑、神圣时刻 |
| 极高 | 几乎全是地面 | 俯瞰、全知 | 鸟瞰城市、战略视角 |

## 第十二章：常见透视错误与修正

### 12.1 错误1：平行线不汇聚

**问题**：同一方向的平行线没有汇聚到消失点
**修正**：确定消失点位置，所有同方向平行线必须指向该点

### 12.2 错误2：地平线不一致

**问题**：同一场景的不同物体地平线位置不同
**修正**：场景中所有物体共享同一地平线（除非有特殊镜头效果）

### 12.3 错误3：仰拍看到顶面

**问题**：仰拍角度却能看到物体顶面
**修正**：仰拍时只能看到物体底面，顶面被遮挡

### 12.4 错误4：俯拍看到底面

**问题**：俯拍角度却能看到物体底面
**修正**：俯拍时只能看到物体顶面

### 12.5 错误5：倾斜面消失点在地平线上

**问题**：楼梯、斜坡等倾斜面的消失点放在地平线上
**修正**：
- 向上的斜面 → 消失点在地平线**上方**
- 向下的斜面 → 消失点在地平线**下方**

# 反打镜头（过肩镜头）AI提示词指南

**创建时间**: 2024-12-30  
**适用场景**: 对话场景、互动场景、情感交流场景

---

## 📖 定义

### 反打镜头（有人物场景）

**反打镜头**（Shot-Reverse-Shot）= **过肩镜头**（Over-the-Shoulder Shot, OTS）

- **核心特征**: 从一个角色肩膀后方拍摄另一个角色
- **画面构成**: 前景角色（虚化）+ 后景角色（清晰）
- **典型用途**: 对话场景中交替展示两个角色的视角
- **必要条件**: 场景中必须有两个或以上的人物

### 反向镜头（无人物场景）

**反向镜头**（Reverse Angle Shot / Reverse Shot）

- **核心特征**: 从相反方向拍摄同一空间
- **画面构成**: 环境 + 空间关系
- **典型用途**: 空间展示、环境切换、建立空间连续性
- **必要条件**: 不需要人物，只需要空间或物体

## 🎬 标准提示词格式

### 一、反打镜头（有人物）

#### 中文提示词模板

```
过肩镜头，从[角色A]肩膀后方拍摄，[角色B]位于画面中央，
[角色A]的肩膀和后脑勺占据画面[左/右]侧前景虚化，
[角色B]面向镜头，表情[描述]，两人距离约[X]米
```

#### 英文提示词模板

```
over-the-shoulder shot from behind [Character A]'s shoulder,
[Character B] at the center of the frame,
[Character A]'s shoulder and back of head occupying [left/right] foreground with shallow depth of field,
[Character B] facing camera with [expression], distance between them about [X] meters
```

---

### 二、反向镜头（无人物）

#### 中文提示词模板

```
反向镜头，从[位置B]拍摄[位置A]，
[环境描述]，[光影描述]，[空间关系]
```

#### 英文提示词模板

```
reverse angle shot, from [Location B] looking toward [Location A],
[environment description], [lighting description], [spatial relationship]
```

## 🎯 关键要点

### 一、反打镜头（有人物）

### 1. 前景角色（虚化）

**必须包含的元素**：
- ✅ 肩膀（shoulder）
- ✅ 后脑勺（back of head）
- ✅ 虚化效果（shallow depth of field / blurred）

**可选元素**：
- 手臂（arm）
- 头发（hair）
- 服装细节（clothing details）

### 2. 后景角色（清晰）

**必须包含的元素**：
- ✅ 位于画面中央（at the center of the frame）
- ✅ 面向镜头（facing camera）
- ✅ 表情描述（expression）

**可选元素**：
- 眼神（gaze）
- 动作（action）
- 情绪（emotion）

### 3. 空间关系

**必须包含的元素**：
- ✅ 两人距离（distance between them）
- ✅ 前景角色位置（left/right foreground）

**建议距离**：
- 亲密对话：1-1.5米
- 正常对话：2-2.5米
- 对峙场景：3-4米

---

### 二、反向镜头（无人物）

### 1. 空间方向

**必须包含的元素**：
- ✅ 拍摄起点（from [Location B]）
- ✅ 拍摄目标（looking toward [Location A]）
- ✅ 空间延伸方向（extending forward/backward）

**关键要点**：
- 明确"从哪里拍"和"拍向哪里"
- 使用方向性词汇（forward/backward, toward/away from）
- 保持空间逻辑一致性

### 2. 环境描述

**必须包含的元素**：
- ✅ 空间类型（corridor, street, room, tunnel）
- ✅ 关键物体（door, window, car, furniture）
- ✅ 空间尺度（20 meters, towering, narrow）

**可选元素**：
- 墙壁纹理（circuit patterns, brick walls）
- 地面材质（concrete, wooden floor）
- 天花板特征（exposed pipes, chandeliers）

### 3. 透视与光影

**必须包含的元素**：
- ✅ 透视效果（narrowing toward distance, converging lines）
- ✅ 光影描述（top lighting, backlit, sunset light）
- ✅ 色调氛围（cool color tone, warm glow, dim）

**建议组合**：
- 一点透视 + 顶光 = 走廊/隧道场景
- 两点透视 + 侧光 = 街道/房间场景
- 逆光 + 剪影效果 = 戏剧性场景

## ⚠️ 常见错误

### 一、反打镜头常见错误（有人物）

#### 错误1：前景角色过于清晰

❌ **错误**：
```
过肩镜头，从晋安肩膀后方拍摄，晋安的肩膀清晰可见...
```

✅ **正确**：
```
过肩镜头，从晋安肩膀后方拍摄，晋安的肩膀和后脑勺占据画面左侧前景虚化...
```

#### 错误2：缺少空间关系

❌ **错误**：
```
过肩镜头，从晋安肩膀后方拍摄，林溪面向镜头
```

✅ **正确**：
```
过肩镜头，从晋安肩膀后方拍摄，林溪位于画面中央，两人距离约2米
```

#### 错误3：使用电影术语而非摄影术语

❌ **错误**：
```
(over-the-shoulder shot:1.3), shot-reverse-shot
```

✅ **正确**：
```
over-the-shoulder shot from behind Jin'an's shoulder
```

---

### 二、反向镜头常见错误（无人物）

#### 错误1：方向描述不清

❌ **错误**：
```
反向镜头，走廊，蓝色门
```

✅ **正确**：
```
反向镜头，从走廊尽头拍摄入口方向，走廊向后延伸，入口处是昏暗的门框
```

**说明**：必须明确"从哪里拍"和"拍向哪里"

---

#### 错误2：缺少透视描述

❌ **错误**：
```
反向镜头，从窗户拍摄门口，房间内部
```

✅ **正确**：
```
反向镜头，从窗户位置拍摄门口方向，
两侧墙壁向远处收窄，门框在画面中央
```

**说明**：反向镜头需要强调透视效果，帮助AI理解空间关系

---

#### 错误3：光影不一致

❌ **错误**：
```
镜头1：顶光照明，冷色调
镜头2（反向）：侧光照明，暖色调
```

✅ **正确**：
```
镜头1：顶光照明，冷色调
镜头2（反向）：顶光照明，冷色调（保持一致）
```

**说明**：反向镜头应保持光源和色调一致，除非有特殊剧情需要

## 💡 进阶技巧

### 1. 调整前景占比

**前景占比小**（10-20%）：
```
晋安的肩膀占据画面左侧边缘前景虚化
```
→ 更多空间展示后景角色

**前景占比大**（30-40%）：
```
晋安的肩膀和后脑勺占据画面左侧1/3前景虚化
```
→ 更强的包围感和代入感

### 2. 调整虚化程度

**轻度虚化**：
```
前景轻微虚化（mild blur）
```
→ 前景角色轮廓清晰

**重度虚化**：
```
前景深度虚化（deep blur / bokeh effect）
```
→ 前景角色完全模糊

### 3. 添加动态元素

**前景角色动作**：
```
晋安的肩膀占据画面左侧前景虚化，手臂缓慢抬起
```

**后景角色动作**：
```
林溪位于画面中央，缓慢转头看向晋安
```

## 📊 使用频率建议

### 一、反打镜头（有人物）

| 场景类型 | 使用频率 | 备注 |
|---------|---------|------|
| 对话场景 | 高（50-70%） | 标准配置 |
| 对峙场景 | 中（30-50%） | 配合其他角度 |
| 情感交流 | 高（60-80%） | 强化情感连接 |
| 动作场景 | 低（10-20%） | 不适合快速动作 |

### 二、反向镜头（无人物）

| 场景类型 | 使用频率 | 备注 |
|---------|---------|------|
| 空间展示 | 高（60-80%） | 建立空间感的标准手法 |
| 探索场景 | 高（70-90%） | 让观众理解空间布局 |
| 追逐场景 | 中（30-50%） | 配合运动镜头 |
| 静态场景 | 低（10-20%） | 避免过度使用 |

### 三、组合使用建议

| 组合方式 | 适用场景 | 效果 |
|---------|---------|------|
| 反打镜头 + 反向镜头 | 角色进入新空间 | 先展示空间（反向），再展示角色反应（反打） |
| 反向镜头 + 全景 | 空间探索 | 先展示空间两端（反向），再展示全貌（全景） |
| 反打镜头 + 特写 | 情感高潮 | 先展示对话（反打），再聚焦情绪（特写） |

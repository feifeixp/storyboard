## 1️⃣ **布料材质类型**

### **天然纤维**

| 材质 | 英文 | 特征描述 |
|------|------|----------|
| **丝绸** | silk | 柔软顺滑，有光泽，垂坠感好 |
| **真丝** | pure silk | 天然光泽，轻盈飘逸 |
| **桑蚕丝** | mulberry silk | 细腻柔滑，温润光泽 |
| **棉** | cotton | 柔软透气，哑光质感 |
| **棉麻** | cotton linen | 自然肌理，略带褶皱感 |
| **麻** | linen | 粗糙质感，天然褶皱 |
| **羊毛** | wool | 温暖厚实，略带绒感 |
| **羊绒** | cashmere | 柔软细腻，轻盈保暖 |
| **天鹅绒** | velvet | 绒面丰厚，光泽变化丰富 |

### **特殊面料**

| 材质 | 英文 | 特征描述 |
|------|------|----------|
| **纱** | gauze / chiffon | 轻薄透明，飘逸朦胧 |
| **欧根纱** | organza | 挺括透明，有轻微光泽 |
| **蕾丝** | lace | 镂空花纹，精致优雅 |
| **缎面** | satin | 高光泽，顺滑细腻 |
| **丝绒** | velour | 柔软绒面，温暖质感 |
| **雪纺** | chiffon | 轻盈飘逸，半透明 |
| **塔夫绸** | taffeta | 挺括有型，带有轻微沙沙声感 |
| **锦缎** | brocade | 华丽织花，厚重有质感 |

### **现代合成材料**

| 材质 | 英文 | 特征描述 |
|------|------|----------|
| **皮革** | leather | 光滑或粗糙纹理，挺括有型 |
| **仿皮** | faux leather | 光滑表面，人造质感 |
| **尼龙** | nylon | 光滑防水，略带光泽 |
| **涤纶** | polyester | 挺括抗皱，光泽稳定 |
| **氨纶/弹力布** | spandex / stretch fabric | 贴身弹性，顺滑光泽 |

## 2️⃣ **材质结构特征**

### **编织方式**

| 特征 | 英文 | 视觉效果 |
|------|------|----------|
| **平纹** | plain weave | 简洁均匀，表面平整 |
| **斜纹** | twill weave | 对角纹理，立体感强 |
| **缎纹** | satin weave | 光滑光亮，反光强 |
| **针织** | knit | 柔软弹性，纹理细腻 |
| **提花** | jacquard | 织入图案，立体花纹 |
| **网眼** | mesh | 镂空透气，网格状 |
| **毛圈** | terry cloth | 柔软蓬松，吸水性强 |

### **厚度与密度**

| 描述 | 英文 | 适用场景 |
|------|------|----------|
| **轻薄** | lightweight / sheer | 夏装、纱衣、内衬 |
| **中等厚度** | medium weight | 常规服装 |
| **厚重** | heavyweight | 外套、冬装 |
| **密织** | tightly woven | 挺括有型 |
| **稀疏** | loosely woven | 透气轻盈 |

### **透明度**

| 描述 | 英文 | 视觉效果 |
|------|------|----------|
| **完全不透明** | opaque | 无透光，遮挡性强 |
| **半透明** | semi-transparent | 隐约透光，朦胧感 |
| **透明** | transparent / sheer | 清晰透视，轻盈飘逸 |

| **珍珠光泽** | pearlescent | 珍珠般变幻光泽 |

| **镜面光泽** | glossy / mirror-like | 极强反光，如镜面般光滑 |

| 效果 | 英文 | 视觉描述 |
|------|------|----------|

| 纹理 | 英文 | 视觉/触感 |
|------|------|-----------|

## 9️⃣ **常见服装风格组合**

### **古风 / 汉服**

**推荐材质组合：**
```
丝绸 + 真丝纱 + 缎面
silk + gauze + satin

光泽：柔光、缎面光泽
垂坠：自然垂坠、飘逸感
装饰：暗纹绣、滚边
版型：修身上衣 + 飘逸下摆
```

---

### **现代时尚 / 职业装**

**推荐材质组合：**
```
棉麻 + 羊毛 + 涤纶混纺
cotton linen + wool + polyester blend

光泽：哑光或柔光
垂坠：挺括有型
装饰：简洁镶边、纽扣
版型：修身剪裁、H型轮廓
```

---

### **晚礼服 / 华丽风**

**推荐材质组合：**
```
缎面 + 蕾丝 + 天鹅绒
satin + lace + velvet

光泽：高光泽、珍珠光
垂坠：流畅垂坠、拖地长度
装饰：珠绣、亮片、褶边
版型：修身或A字型
```

---

### **休闲 / 运动风**

**推荐材质组合：**
```
棉 + 针织 + 弹力布
cotton + knit + stretch fabric

光泽：哑光
垂坠：柔软贴身
装饰：简洁或无装饰
版型：宽松或修身
```

---

### **仙气 / 飘逸风**

**推荐材质组合：**
```
雪纺 + 欧根纱 + 真丝纱
chiffon + organza + silk gauze

光泽：柔光、微光泽
垂坠：飘逸、随风飞扬
装饰：蕾丝边、流苏
版型：A字型、多层叠加
```

---

### **复古 / 中世纪风**

**推荐材质组合：**
```
锦缎 + 天鹅绒 + 厚重丝绸
brocade + velvet + heavy silk

光泽：暗光泽、金属光
垂坠：挺括有型、深度褶皱
装饰：金线绣、立体绣
版型：宫廷式、蓬裙
```

## 📝 **快速查找索引**

### **光泽程度（从低到高）**
哑光 → 柔光 → 缎面光泽 → 珍珠光泽 → 金属光泽 → 镜面光泽

### **厚度（从薄到厚）**
轻薄纱 → 薄丝绸 → 中等棉布 → 厚呢料 → 羊毛外套料

### **透明度（从透到不透）**
透明纱 → 半透雪纺 → 薄棉 → 厚棉 → 完全不透明

### **挺括度（从软到硬）**
柔软雪纺 → 垂坠丝绸 → 普通棉 → 挺括棉麻 → 硬质塔夫绸

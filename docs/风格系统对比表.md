## 💡 使用建议

### 1. 首次使用

**推荐流程**:
1. 查看 `docs/分镜风格使用指南.md`
2. 根据题材类型选择 2-3 个风格
3. 生成小批量测试（9个镜头）
4. 对比效果，选择最佳风格
5. 批量生成完整分镜

### 2. 风格混搭

**不推荐混搭**:
- 同一个项目中使用多种风格会导致视觉不统一
- 建议整个项目使用同一种风格

**例外情况**:
- 回忆场景可以使用不同风格（如黑白电影）
- 梦境场景可以使用特殊风格（如水墨写意）

### 3. 风格与题材匹配

**高度匹配** (推荐):
- 修真题材 → 3D国潮动漫 或 水墨写意
- 二次元题材 → 日式赛璐璐
- 悬疑题材 → 电影超写实 或 黑白电影

**中度匹配** (可用):
- 修真题材 → 数字艺术厚涂
- 二次元题材 → 2D Q版卡通
- 悬疑题材 → 手绘线稿

**低度匹配** (不推荐):
- 修真题材 → 像素艺术
- 二次元题材 → 电影超写实
- 悬疑题材 → 3D 黏土/盲盒
